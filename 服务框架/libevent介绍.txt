1. libevent介绍  一个封装了epoll等的框架，比直接使用epoll要简单。  可以实现3类型功能:    1) 网络收发      对epoll等的封装    2) 定时器      用最小堆实现    3) 信号处理      为了保持外部接口一致，实际是将信号信息，通过socketpair触发      由于信号处理必须小心翼翼，特别是多线程下的处理。因此，不建议使用此功能。      如memcached使用的libevent，但信号处理还是直接调用的原始api      2. 下载安装  官网：www.libevent.org  下载最新源代码, libevent-2.0.21-stable.tar.gz  ./configure  make  make install  可以执行/test/test.sh验证3. 重要的数据结构:  1. struct event    封装的是可读可写时间    event_new()、event_assign()、event_add()、event_del()、event_free()  2. struct event_base    也就是reactor, epoll引擎    event_base_new()、event_base_new_with_config()、event_base_dispatch()、event_base_loop()、event_base_free()      3. struct event_config    为event_base提供配置(配置非常多，仅仅只是为了不必传太多的参数)    event_config_new()、event_config_free()    用例请参考 src/libevent/single_server.cpp
