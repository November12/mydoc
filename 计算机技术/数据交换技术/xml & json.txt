1. 语义描述                 all  Win ！
  虽然xml的标签还可以加属性，扩展性更强一些。但还没有什么xml能描述的，但json描述不了的。
  两者都有很好的可读性。
  可以说，在这个层面上，两者完全一样。
  
2. 数据大小                 json Win ！
  显然，json比xml体积更小，传输的开销肯定比xml要小。
  
3. 编码解码                 all  Lost ！
  两者都需要建立解析树的文档对象，再通过文档对象获取需要的数据。
  两者效率都不高。
  
4. 普及程度                 xml  Win ！
  无疑，xml的普及程度要高很多，仅此一项，就可以将json打得体无完肤。
