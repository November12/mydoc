1. 介绍  cscope和ctags非常的相似。只不过cscope更多的搜索类型和选项。  算是一个加强版的ctags吧。  并且cscope的运行也需要依赖ctags。2. 安装  2.1 下载    官网是http://cscope.sourceforge.net/，但貌似打不开了，下载位置查baidu吧    我是在sourceforge上查找到的，      2.2 安装    非常简单，没有特殊步骤:    ./configure    make    make install      2.3 验证    cscope --help    vim中执行:cs (vim只有编译时加上了--enable-cscope选项，才支持cscope)3. 使用方法  3.1 生成一个cscope数据库    源代码根目录下执行 cscope -Rbq。会生成三个文件cscope.in.out，cscope.out，cscope.po.out。(不用打开，都看不懂的)      3.2 让vim把cscope.out加载进去    .vimrc里添加:    if has("cscope")        set csprg=/usr/bin/cscope        set csto=1        set cst         set nocsverb        if filereadable("cscope.out")            cs add cscope.out        elseif $CSCOPE_DB  != ""            cs add $CSCOPE_DB        endif        set csverb    endif    nmap <C-\>s :cs find s <C-R>=expand("<cword>")<CR><CR>    nmap <C-\>g :cs find g <C-R>=expand("<cword>")<CR><CR>    nmap <C-\>c :cs find c <C-R>=expand("<cword>")<CR><CR>    nmap <C-\>t :cs find t <C-R>=expand("<cword>")<CR><CR>    nmap <C-\>e :cs find e <C-R>=expand("<cword>")<CR><CR>    nmap <C-\>f :cs find f <C-R>=expand("<cfile>")<CR><CR>    nmap <C-\>i :cs find i <C-R>=expand("<cfile>")<CR>$<CR>    nmap <C-\>d :cs find d <C-R>=expand("<cword>")<CR><CR>      3.3 举例:    ctrl-]    ctrl-t        s: 查找所有出现的地方    g: 查找定义    d: 查找本函数调用的函数    c: 查找调用本函数的函数    t: 查找指定的字符串    e: 查找egrep模式，相当于egrep功能，但查找速度快多了    f: 查找并打开文件，类似vim的find功能    i: 查找包含本文件的文
