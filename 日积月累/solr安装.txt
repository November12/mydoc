JDK：JDK 1.6.0_16Tomcat: 6.0.201. 安装JDK  1.1 下载:     http://java.sun.com/javase/downloads/index.jsp 的 jdk-6u16-linux-i586-rpm.bin  1.2 安装    # sh /home/apps/jdk-6u16-linux-i586-rpm.bin (默认会安装在/usr/java目录下)   验证一下:    # java -version   验证配置:    # vi /etc/profile    JAVA_HOME=/usr/java/jdk1.6.0_16    CLASSPATH=.:$JAVA_HOME/lib/tools.jar    PATH=$JAVA_HOME/bin:$PATH    export JAVA_HOME CLASSPATH PATH    2. 安装Tomcat  2.1 下载    http://tomcat.apache.org/download-60.cgi 的 apache-tomcat-6.0.20.tar.gz    注意: 下载二进制包，不要下载源码包  2.2 安装    # tar zxvf apache-tomcat-6.0.20.tar.gz    # cp -a apache-tomcat-6.0.20 /usr/local/apache-tomcat-6.0.20    # cd /usr/local/apache-tomcat-6.0.20/bin    # ./startup.sh   验证一下:    http://10.15.107.155:8080/3. 安装Solr  3.1 下载    http://www.apache.org/dyn/closer.cgi/lucene/solr/ 的 apache-solr-3.6.2.gz    3.2 安装    tar xzvf apache-solr-3.6.2.gz     cp apache-solr-3.6.2/example/webapps/solr.war /usr/local/apache-tomcat-6.0.36/webapps/    mv apache-solr-3.6.2 /usr/local/    vi /usr/local/apache-tomcat-6.0.36/conf/Catalina/localhost/solr.xml       <Context docBase="/usr/local/apache-tomcat-6.0.36/webapps/solr.war" debug="0" crossContext="true">        <Environment name="solr/home" type="java.lang.String" value="/usr/local/apache-solr-3.6.2/example/solr" override="true"/>      </Context>    /usr/local/apache-tomcat-6.0.36/bin/catalina.sh stop    /usr/local/apache-tomcat-6.0.36/bin/catalina.sh start    4. 配置SolrCloud  4.0 预备    tomcat的配置文件，Catalina/localhost/solr.xml里, 需要设置override = "false", 这是避免solr.war包里的默认配置会覆盖这里的配置。        4.1 安装ZooKeeper, 这里地址是10.15.107.176:2181    尽管可以通过使用-DzkRun，启动内置的ZooKeeper，但从统一管理的角度来看，还是外置一个ZooKeeper比较合适  4.2 配置执行    无需多余的配置，在第一台机器(作为leader)上启动tomcat之前，执行:      export JAVA_OPTS="-Dbootstrap_confdir=/home/solr/solr-4.1.0/example/solr/collection1/conf -Dcollection.configName=myconf -DzkHost=10.15.107.176:2181 -DnumShards=2"      leader并非永远的leader，但启动时，总需要一个标准    后面的机器依次执行, 并启动tomcat:      export JAVA_OPTS="-DzkHost=10.15.107.176:2181 -DnumShards=2"      顺序是:        shard2:1        shard1:2        shard2:2        ...      这之后，顺序就固定了，因此第一次是很重要的          4.3 验证    访问页面：http://host:8983/solr/#/~cloud 看看集群状态
