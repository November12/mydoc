1. 下载  官网(http://oozie.apache.org/)下载最新的安装包，oozie-3.3.1-distro.tar.gz  Oozie运行还需要Mysql, Hadoop的库，也一并下载安装。 (执行脚本bin/oozie-setup.sh安装)  2. 配置选项  修改conf/oozie-site.xml, 主要是Mysql相关配置  <property>    <name>oozie.service.JPAService.create.db.schema</name>    <value>true</value>    <description>        Creates Oozie DB.        If set to true, it creates the DB schema if it does not exist. If the DB schema exists is a NOP.        If set to false, it does not create the DB schema. If the DB schema does not exist it fails start up.    </description>  </property>  <property>      <name>oozie.processing.timezone</name>      <value>GMT+0800</value>  </property>  <property>     <name>oozie.service.JPAService.jdbc.driver</name>     <value>com.mysql.jdbc.Driver</value>                <description>       JDBC driver class.     </description>   </property>  <property>     <name>oozie.service.JPAService.jdbc.url</name>     <value>jdbc:mysql://localhost:3306/oozie</value>     <description>       JDBC URL.     </description>   </property>   <property>     <name>oozie.service.JPAService.jdbc.username</name>     <value>oozie</value>     <description>       DB user name.     </description>   </property>   <property>    <name>oozie.service.StoreService.jdbc.password</name>    <value>oozie</value>  </property>  3. 在mysql中创建db oozie4. 执行脚本, 初始化数据库  bin/ooziedb.sh create -sqlfile oozie.sql -run5. 修改hadoop集群配置(如果oozie也使用hadoop权限运行此步骤可以省略)  在hadoop/conf/core-site.xml中添加  <property>     <name>hadoop.proxyuser.[oozie运行用户].hosts</name>     <value>安装oozie服务器ip</value>  </property>  <property>     <name>hadoop.proxyuser.[oozie运行用户].groups</name>     <value>oozie运行用户</value>  </property>  重启hadoop6. 启动oozie  /bin/oozie-start.sh7. 查看web控制台验证  http://10.15.107.64:11000/oozie/
