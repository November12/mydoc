Java方面:* 应用服务器  Tomcat                  75%  Oracle Weblogic         10%  JBoss AS                5%  IBM Websphere           5%  可以看出，尽管Tomcat有诸多不足，但以其成熟和稳定，依然占有绝对的地位。  * Java Web开发框架  Spring MVC              36%  structs                 26%  不使用/自行开发         22%  SSH依然是主流，也有相当一部分不使用开源框架  * Java开发工具  Eclipse                 90%  估计很长很长一段时间内，无人能替代Eclipse  * Web前端表现技术  HTML + CSS              48%  Ajax                    40%  HTML5                    6%  Flash                    2%  主流技术还是HTML/CSS 和 Ajax。HTML5虽然很火，但目前使用的还不多(相信明年会多很多)。而Flash基本已经没人用了。  * 数据库  MySql                   38%  Oracle                  30%  SQL Server              20%  DB2                      6%  值得一提的是，选择何种数据库，和访问量没有太大关系。  当访问量>1000万次的时候，更多的人选择Mysql(50%) 和 Oracle(33%)。(无人选择Sql Server)
